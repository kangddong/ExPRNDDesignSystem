# ExPRNDDesignSystem

평소 디자인시스템을 접해볼 기회가 없어서 궁금하기만한 미지의 영역이라고 생각했는데,

PRND는 디자인시스템을 어떻게 구성했는지 Step by Step으로 세션이 진행을 진행해주어서 예제 코드를 완성해보고싶었습니다.
